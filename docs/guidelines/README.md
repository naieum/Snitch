## The Philosophy

We make a tool that developers actually use. That means:

1. **We prove everything.** No findings without proof. No vague warnings. No AI panic.
2. **We talk like humans.** Not compliance jargon. Not marketing speak. Just straight talk about what's actually wrong and how to fix it.
3. **We do security right.** Our tool reviews AI-generated code. We set the standard for what "good" means.

Reading one of these files shouldn't feel like homework. It should feel like a conversation with someone who actually cares about making your job easier.

Got questions? That's what issues are for.
